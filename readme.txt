# Mentalize 🧠💬

**Mentalize** é uma plataforma de apoio psicológico acessível, criada por estudantes de Psicologia com supervisão profissional. Nosso objetivo é democratizar o acesso à saúde mental, especialmente para jovens de baixa renda, pessoas com ansiedade e indivíduos que enfrentam burnout.

---

## 🌐 Tecnologias Utilizadas
- HTML5
- CSS3 (Responsivo + animações)
- JavaScript (efeitos de rolagem)

---

## 📄 Páginas

### `index.html`
Página inicial com apresentação da plataforma, valores e missão.

### `planos.html`
Modelos de atendimento disponíveis:
- Consulta individual por hora
- Plano institucional para escolas e universidades
- Plano empresarial para empresas

### `contato.html`
Formulário de contato + botão direto para atendimento via WhatsApp.

---

## 🧩 Funcionalidades
- Design moderno e responsivo
- Animações suaves na rolagem
- Botão para atendimento automatizado via WhatsApp

---

## 🛠️ Em desenvolvimento futuro:
